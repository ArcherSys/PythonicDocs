linecache module¶

Cache lines from files.

This is intended to read lines from modules imported – hence if a filename is not found, it will look down the module search path for a file by that name.

linecache.checkcache(filename=None, orig_checkcache=<function checkcache at 0x01833ED0>)¶

Extend linecache.checkcache to preserve the <pyshell#...> entries

Rather than repeating the linecache code, patch it to save the <pyshell#...> entries, call the original linecache.checkcache() (skipping them), and then restore the saved entries.

orig_checkcache is bound at definition time to the original method, allowing it to be patched.
