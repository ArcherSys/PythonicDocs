tkinter.test.runtktests module¶

Use this module to get and run all tk tests.

tkinter tests should live in a package inside the directory where this file lives, like test_tkinter. Extensions also should live in packages following the same rule as above.

tkinter.test.runtktests.get_tests(text=True, gui=True, packages=None)[source]¶

Yield all the tests in the modules found by get_tests_modules.

If nogui is True, only tests that do not require a GUI will be returned.

tkinter.test.runtktests.get_tests_modules(basepath='C:\\Python34\\lib\\tkinter\\test', gui=True, packages=None)[source]¶

This will import and yield modules whose names start with test_ and are inside packages found in the path starting at basepath.

If packages is specified it should contain package names that want their tests collected.
