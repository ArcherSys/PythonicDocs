compileall module¶

Module/script to byte-compile all .py files to .pyc (or .pyo) files.

When called as a script with arguments, this compiles the directories given as arguments recursively; the -l option prevents it from recursing into directories.

Without arguments, if compiles all modules on sys.path, without recursing into subdirectories. (Even though it should do so for packages – for now, you’ll have to deal with packages separately.)

See module py_compile for details of the actual byte-compilation.

compileall.compile_dir(dir, maxlevels=10, ddir=None, force=False, rx=None, quiet=False, legacy=False, optimize=-1)[source]¶

Byte-compile all modules in the given directory tree.

Arguments (only dir is required):

dir: the directory to byte-compile maxlevels: maximum recursion level (default 10) ddir: the directory that will be prepended to the path to the

file as it is compiled into each byte-code file.

force: if True, force compilation, even if timestamps are up-to-date quiet: if True, be quiet during compilation legacy: if True, produce legacy pyc paths instead of PEP 3147 paths optimize: optimization level or -1 for level of the interpreter
